3️⃣ No Render:
   - Clique em "New" → "Web Service"
   - Conecte ao seu repositório
   - Configure:
       Build Command: npm install
       Start Command: npm start
       Environment: Node
       Free Plan ✅

4️⃣ Vá em Settings → Environment → Add Environment Variable:
       Key: DISCORD_TOKEN
       Value: <seu token do Discord>

5️⃣ Deploy e aguarde:
   Logs → “✅ Logado como SeuBot#1234”
